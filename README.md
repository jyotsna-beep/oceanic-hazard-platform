# 🌊 Oceanic Hazard Intelligence Platform
### AI-Powered Disaster Monitoring & Early Warning System

An intelligent full-stack platform designed to **detect, classify, and disseminate oceanic hazard alerts in real time** using machine learning, data fusion, and automated communication pipelines.

This project was developed by **Team CoralCrew** for **Smart India Hackathon (SIH25139)** under the **Disaster Management Theme**.

The system integrates **machine learning models, real-time data ingestion, and a scalable web platform** to provide early warnings and improve disaster response efficiency.

---

## 📌 Problem Statement

Natural disasters such as **cyclones, floods, storm surges, and coastal hazards** often cause devastating damage due to **delayed detection and slow information dissemination**.

Existing systems suffer from:

- Limited real-time monitoring
- Delays in hazard alerts
- Poor integration of multi-source data
- Low public engagement with disaster alert platforms

This platform aims to solve these problems by combining:

- **AI-driven hazard detection**
- **Real-time data aggregation**
- **Automated alert dissemination**
- **Interactive monitoring dashboards**

---

## 🎯 Project Objectives

- Detect oceanic hazards using **machine learning models**
- Fuse **multiple real-time data sources**
- Provide **early warning alerts**

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/jyotsna-beep/oceanic-hazard-platform.git
cd oceanic-hazard-platform
```
Install dependencies:

npm install

Run backend:

npm start

Run frontend:

npm run dev

Run ML models:

python train_model.py
